Description: This package includes the the four algoirthms implemented in [1] for multi-label crowdsourcing learning. A readme file and example data is available in the package.  

References:S.-Y. Li, Y. Jiang,  N.V. Chawla, and Z.-H. Zhou. Multi-label learning from crowds. IEEE Transactions on Knowledge and Data Engineering 31(7), 1369{1382 (2019)


Requirement: The package was developed with MATLAB.
